# Livepeer Token (LPT)

Livepeer Token (LPT) is the native [ERC-20](https://ethereum.org/en/erc-20/) token that secures the Livepeer Network and enables community-driven governance. 

Holding LPT allows participants to earn fees and rewards, as well as influence the network’s direction through governance votes. 

LPT aligns incentives for all network actors to keep Livepeer cost-effective, efficient, secure, and reliable for real-time video & AI capabilities. 

This “stake-for-access” model means the more LPT you stake, the more work your node can receive in the network, bootstrapping a marketplace for video infrastructure

# Task RFP-M12: Measurable engagement

**Missing idea:** M12  
**RFP deliverable:** (iii) V1 Documentation Live — **Outcome**

---

## Objective

Define and implement “measurable **engagement**” for the docs site so that the (iii) outcome “measurable engagement” is tracked and reported.

## Scope

- RFP (iii) Outcome: “measurable **engagement**”
- Analytics “not verified”; “engagement” as a success metric is not explicitly tracked
- Deliverable: define engagement metrics (e.g. page views, search queries, time on page, CTA clicks), implement tracking, and document where to view them

## Deliverables

- Short “Docs engagement metrics” note: which metrics count as “engagement” and where they are viewed (e.g. Mintlify analytics, GA, custom dashboard)
- Verification that analytics/tracking is enabled and capturing at least one engagement metric
- Optional: baseline or target for “measurable” (e.g. “we measure X; baseline established on [date]”)

## Acceptance criteria

- [ ] At least one engagement metric is defined and measurable
- [ ] Tracking is in place and verified (or documented as “pending” with owner)
- [ ] RFP outcome “measurable engagement” is marked complete in the gaps report when done

# Task RFP-M16: Ecosystem integration (website, explorer, governance, dashboards)

**Missing idea:** M16  
**RFP deliverable:** Desired outcome (success criteria)

---

## Objective

Verify and document how the docs **integrate with the broader Livepeer ecosystem** (website, explorer, governance, dashboards) so the success criterion is no longer “Unknown.”

## Scope

- Desired outcome: “Integrates seamlessly with the broader Livepeer ecosystem (**website, explorer, governance, dashboards**)”
- Report: “Unknown.” No checklist for docs ↔ website, docs ↔ explorer, docs ↔ governance, docs ↔ dashboards
- Deliverable: for each surface (website, explorer, governance, dashboards), state how docs integrate (links, nav, embeds, CTAs) or what’s missing

## Deliverables

- Checklist or short doc: “Ecosystem integration”
  - **Website:** how docs are linked from livepeer.org (nav, footer, CTAs)
  - **Explorer:** any link or reference from explorer to docs (or vice versa)
  - **Governance:** link from governance portal to docs (e.g. “How to participate”, protocol docs)
  - **Dashboards:** any dashboard that links to docs or embeds help
- For each: “Done” with evidence (URL/screenshot) or “Not yet” with owner/next step
- Update gaps report so M16 is “Done” or “In progress” with link to this checklist

## Acceptance criteria

- [ ] All four surfaces (website, explorer, governance, dashboards) are explicitly checked
- [ ] Integration (or gap) is documented for each
- [ ] RFP success criterion “integrates with ecosystem” is no longer Unknown in the assessment

# Task RFP-M4: Delegators — integrate new video content

**Missing idea:** M4  
**RFP deliverable:** (i) Present New Documentation Strategy

---

## Objective

Integrate **new** video content into delegator documentation to make it easy to delegate, per the RFP (not only a "diagram/video pointer").

## Scope

- RFP (i): "Delegators: **integrate new video content** to make it easy to delegate"
- Task list has "diagram/video pointer" in Phase 3 outputs but not "integrate new video content" as a distinct requirement
- "New" may mean newly produced for this RFP or newly added to the docs (identify and document)

## Deliverables

- List of existing or planned video assets for delegators (what, where they live, license)
- Integration into delegator docs: embed or link from relevant pages (e.g. stake/unstake, safety, fee/tax)
- Short note on what "new" means for this deliverable (new to docs vs newly produced)

## Acceptance criteria

- [ ] At least one video (or set of clips) is identified for delegator use
- [ ] Delegator docs link to or embed this video content where it helps "make it easy to delegate"
- [ ] RFP requirement "integrate new video content" is documented as done (in report or Notion)
